# Sample Datasets

This folder includes synthetic CSV files for quick API testing.

- `sample_classification.csv`: Binary classification dataset with `target` column.
- `sample_unsupervised.csv`: Numeric feature dataset for KMeans and PCA endpoints.
- `sample_regression.csv`: Numeric regression dataset with `price` as target for `/upload` flow.

You can upload these files directly through Swagger UI at `/docs`.
